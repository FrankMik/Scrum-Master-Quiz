# 🧠 Scrum Master Quiz

Eine kleine Webanwendung, mit der Nutzer ihr Wissen über Scrum und die Rolle des Scrum Masters in Form eines Multiple-Choice-Quiz testen können.

## 🎯 Ziel des Projekts

- Festigung von Scrum-Grundlagen (Events, Artefakte, Rollen).
- Übung in der Frontend-Webentwicklung (z.B. React/Vanilla JS, State-Handling, UI-Logik).
- Vorbereitung auf Prüfungen oder Zertifizierungen mit spielerischem Lernen.

## ✨ Features

- Multiple-Choice-Fragen zu Scrum (Rollen, Events, Artefakte, Prinzipien).
- Punktestand und Auswertung am Ende des Quiz.
- Direkte Rückmeldung, ob eine Antwort richtig oder falsch war.
- Möglichkeit, das Quiz erneut zu starten und weiter zu üben.
- Responsives Layout für Desktop und Mobile (je nach deiner Implementation).
